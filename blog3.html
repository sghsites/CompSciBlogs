What is a Variable?

Here is a variable used in a print function

Variables are like shortcuts, they are used to refer to a statement or argument. This is shown by their two parts, the name and the argument / statement they represent. In this case a variable is used to reference an input.



Variable Names

It might be self-explanatory to name variables, but beginners still make a lot of mistakes. For example, the name bob is valid but 1bob is not because variables can't have special characters in front of them. This is because variables have a 'dress' type code for naming. This code says that variables can't have special characters in the first spot in the name but can after a letter. That might make sense but variables can have underscores anywhere in the name including before and after the variable itself like _APP or _APP_
