How To Use Functions


Defining a Python Function

Here is a simple function that shows a greeting

Each function has two parts, definition and body. The function definition is what names the function. In this case the function is named greet. It is also preceded by the keyword def , which means define. The function body is what the function will do. In this function the body says to print the word, hi.



Calling a Function

To call a function you have to write the function’s name and followed by the information it needs. For example, This calls the greet function and since we don’t need a parameter the () is blank. greet() This will return the script defined by the function we wrote earlier. hi



Passing Information To a Function

Say we want to greet someone by their name. We would have to specify their name in the () as follows def greet(name): The name is called a parameter. You can use the parameters to specify variables in a function.

This concludes the second blog of 3 about writing python. In this blog we covered how to use functions. First, we discussed the syntax / defining a function. Second, we discussed how to call a function. Third, we discussed how to pass information into a function.
